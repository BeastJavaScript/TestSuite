Class: TestCase

Defined in: src\TestCase.coffee

Overview

This class provides an extremely readable testing framework for testing with CoffeeScript and Javascript in General. It was built using CoffeeScript.

You must keep this header in all copy of this Source Code.

# (void) assertEquals(test, value)

This will test if two values are equal. This will do a deep Object Comparison on Objects and Arrays. Be careful of Objects that might contain circular references. The Application will handle it just fine but it will result in a warning. These Object will not neccesarily be equal

Parameters:

  • test ( Object,String,Number,Boolean )
  • value ( Object,String,Number,Boolean )

# (void) assertArrayEquals(array1, array2)

This will test if two arrays are equal. This will not do a deep object test on the array. It will only check the first level of the array

# (void) assertObjectEquals(obj1, obj2)

This is essential the same as TestCase#assertEquals. This is just a more reader friendly form that is better at documenting your intention.

# (void) assertSame(test, value)

This will check if two objects are the same objects. It will also test if two primitives are equal.

Parameters:

  • test ( Object,String,Number,Boolean )
  • value ( Object,String,Number,Boolean )

# (void) assertSimilar(test, value)

This will test if two values are similar. This is mainly for primitive values when comparing a string and a number. When used on a object it is essentially the same as TestCase#assertSame. This will also ignore casing when comparing strings

Examples:

5=='5' #true
'PizzaMan' == 'pizzaman' #true

Parameters:

  • test ( Object,String,Number,Boolean )
  • value ( Object,String,Number,Boolean )

# (void) assertFunction(func)

This will check if a value is a function. This can be useful with callbacks testing.
